Business Logic Requirements:
1. Average Grade:
   - Weighted average formula:
     sum(grade * weight) / sum(weight)
   - If no grades exist, show “No grades available”.

2. Class Comparison:
   - difference = studentAverage - classAverage
   - Above Average if difference > 5
   - Close to Average if difference between -5 and 5
   - Below Average if difference < -5

3. Weekly Assignments:
   - Upcoming assignments are assignments with due date from today to 7 days ahead.
   - Overdue assignments are assignments with due date before today and not completed.
   - Reminder appears when assignment is due within 24 hours.

4. Submission Rate:
   - submissionRate = submittedAssignments / totalAssignments * 100
   - Late submissions should still count as submitted, but should be shown separately.
   - Missed submissions should reduce the rate.
   - If actual rate < target rate, show warning.
   - If actual rate < 60%, show critical alert.
